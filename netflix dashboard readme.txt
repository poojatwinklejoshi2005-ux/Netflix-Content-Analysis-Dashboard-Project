Netflix Data Analysis Dashboard Project

Project Overview
This project is a data analysis and visualization dashboard based on Netflix dataset
It provides insights into movies and TV shows available on Netflix
The dashboard is created using Tableau (or Power BI if applicable)
It helps understand content distribution, ratings, genres, and trends

Objectives
Analyze Netflix content dataset
Understand distribution of Movies vs TV Shows
Identify most common genres and categories
Study content ratings and audience classification
Visualize trends over time

Dataset Information
Source: Netflix dataset (CSV file)
Key columns include:
Title
Type (Movie/TV Show)
Director
Cast
Country
Date Added
Release Year
Rating
Duration
Genre (Listed In)

Tools Used
Tableau / Power BI (for dashboard creation)
Microsoft Excel / CSV file (for dataset handling)
Data cleaning tools (as required)

Data Cleaning Steps
Removed duplicate records
Handled missing values in key columns
Standardized date formats
Split multi-value columns where required (like genre, cast)
Filtered irrelevant or incomplete entries

Dashboard Features
Total number of Movies and TV Shows
Year-wise content addition trend
Top countries producing Netflix content
Most popular genres
Rating distribution of content
Interactive filters for:
Type (Movie / TV Show)
Country
Year
Rating

Key Insights
Movies are more in number compared to TV Shows
Netflix content increased significantly after specific years
United States and India are major content contributors
Drama and International content are most common genres
TV-MA and TV-14 are common content ratings

Project Outcome
Better understanding of Netflix content distribution
Improved skills in data visualization and dashboard design
Ability to extract meaningful insights from raw data

How to Use
Open the dashboard file in Tableau / Power BI
Apply filters to explore data interactively
Hover over charts to view detailed values

Future Improvements
